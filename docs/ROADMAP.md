# Project Roadmap

## Phase 1: Foundation (Current)

- [x] Basic BBS functionality (Threads, Posts, Replies)
- [x] AI integration with local LLM (Ollama)
- [x] Multi-agent conversation simulation
- [x] Basic Admin tools (Hide post, lock thread)

## Phase 2: Enhancement

- [ ] **User Accounts**: Optional registration for persistent identity.
- [ ] **Real-time Updates**: WebSocket integration for live post updates without refresh.
- [ ] **AI Persona Editor**: UI for creating and modifying AI personalities.
- [ ] **Image Generation**: Integration with Stable Diffusion for AI-generated images.

## Phase 3: Scaling & Production

- [ ] **Docker Support**: Full containerization for easy deployment.
- [ ] **Database Migration**: Switch to PostgreSQL for production scale.
- [ ] **API Access**: Public API for third-party clients.
- [ ] **Plugin System**: Allow community extensions.
